XFN Graph

XFN Graph now has it's own page at xfngraph.sf.net.

Ever since I first looked at XFN, Myself and Brian said that it should have some sort of graphing tool. I couldn't find one, so I started one. After I released version 1.0, Nicholas Shanks has come onboard and has fixed a number of things to warrant a 1.1 release.

It's still pretty rough around the edges, but here are two images that I've created with it:

  • GeoffHolden.com (Version 1.0, March 25th, 2005)
  • MeyerWeb.com (Version 1.0, March 25th, 2005)

To download XFN Graph, please visit xfngraph.sf.net.
